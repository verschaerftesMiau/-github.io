Zusatzregeln:

Bei einer gespielten 3 muss eine Karte an einen Nachbarn abgegeben werden. Wird dies vergessen, bekommt man 2 Strafkarten. Eine 3 darf demnach nicht als letzte Karte gespielt werden.
